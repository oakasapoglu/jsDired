# jsDired

jsDired is a directory editor that runs entirely in your web browser. It is designed to give you a keyboard-centric, list-based interface for managing files and folders on your computer without needing to install native software.

## User Benefits
*   **Zero-Install Management:** Access and edit your local filesystem through a standard web tab.
*   **Privacy-First:** The application runs locally in your browser and uses the **File System Access API** to interact directly with your data—no files are uploaded to a server.
*   **Efficient Workflow:** Optimized for users who prefer a minimalist, list-based view and keyboard shortcuts for navigating, renaming, and organizing directories.
