⚠️ Experimental Tech: Proceed with Caution

These are raw, AI-generated experiments using the browser's File System Access API. While they are incredibly fast and showcase a new way to work with local files, they operate without the safety nets of a native operating system.

Permanent Deletion: There is no "Trash" folder or "Undo" history here. When a file is deleted or overwritten by these scripts, it is gone forever.
AI Logic Risk: These implementations were generated by LLMs. They are optimized for speed and features, not rigorous safety. A logic error could result in unintended mass-deletion.
Direct Write Access: You are granting a browser environment full control over your local data. Use this power carefully.

Strict Recommendation: Only point these tools at isolated, non-critical test directories. Never use them as a primary file manager for your important work.
